基本命令
cd + 目录名 改变目录
cd.. 回到上一级目录
pwd 显示当前所在的路径
clear（Linux）/cls（Windows） 清屏操作
ls 列出当前文件下的所有文件
touch 新建一个文件，例如touch index.js 则会在当前目录下新建一个index.js文件
rm（remove） 删除一个文件
mkdir 新建一个目录，即新建一个文件夹
rm-r 删除一个文件夹，rm-r src 即删除src目录
rm-rf/直接一删到底
mv 移动一个文件，例如 mv index.html test，将index.html文件移动到test中
reset 重新初始化终端/清屏
history查看历史命令
